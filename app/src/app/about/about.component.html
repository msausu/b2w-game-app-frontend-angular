O Jogo

Nossos associados são apaixonados por Guerra nas Estrelas e, com isso, queremos criar um jogo com algumas informações da franquia.

Para permitir que a equipe da frente crie esse aplicativo, queremos desenvolver uma API que contenha os dados dos planetas.

Requisitos

  • A API deve ser REST.
  • Para cada planeta, os seguintes dados devem ser obtidos do banco de dados do aplicativo, sendo inseridos manualmente:
    • Nome
    • Clima
    • Terreno
  • Para cada planeta, também devemos ter o número de aparições nos filmes, que podem ser obtidos através da API pública de Star Wars.

Recursos desejados

  • Adicione um planeta (com nome, clima e terreno)
  • Listar planetas
  • Procura por nome
  • Pesquisa por ID
  • Remover planeta
